# Next.JS Project Engine - Components Documentation

Welcome to the component documentation for Next.JS Project Engine. This library provides reusable, production-ready components built on top of [shadcn/ui](https://ui.shadcn.com/) and [TanStack](https://tanstack.com/) libraries.

## Quick Start

### Installation

All components are included in the project. No additional installation needed.

### Basic Usage

```tsx
import { DataTable } from "@/components/organisms/@tanstack-react-table/data-table";

const columns = [
  createColumn("id", "ID", "id"),
  createSortableColumn("name", "Name", "name"),
];

<DataTable
  tableName="my-table"
  columns={columns}
  data={myData}
/>
```
