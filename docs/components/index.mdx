# Components

Reusable component library built on top of shadcn/ui and TanStack libraries.

## Available Components

- [DataTable](/components/datatable) - Comprehensive data table with sorting, filtering, pagination, and more
- [Form](/components/form) - Reusable form component (coming soon)

## Getting Started

All components are located in `src/components/` and are fully typed with TypeScript. They're designed to be:

- **Reusable** - Drop-in ready with sensible defaults
- **Customizable** - Extensive props for customization
- **Type-safe** - Full TypeScript support
- **Accessible** - WCAG compliant
- **Themeable** - Works with shadcn/ui theming

## Installation

Components use existing dependencies in your project. No additional installation required.

## Usage

```tsx
import { DataTable } from "@/components/organisms/@tanstack-react-table/data-table";
```
